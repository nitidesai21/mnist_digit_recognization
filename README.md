MNIST Handwritten Digit Classification Web App
A full-stack machine learning project that classifies handwritten digits using the MNIST dataset and serves predictions through a simple web interface. It combines data preprocessing, model training, evaluation, and deployment in a practical end-to-end workflow suitable for a portfolio project.

Overview
This project builds a handwritten digit recognizer using the MNIST dataset, a standard benchmark in computer vision and machine learning. The workflow covers loading and preparing image data, training a classification model, testing performance, and integrating the trained model into a Flask-based web application for real-time predictions.

The project is designed to demonstrate both machine learning fundamentals and deployment skills. It is a strong beginner-to-intermediate project for showcasing model development, dataset handling, and application integration on GitHub.

Features
Train a machine learning model on the MNIST handwritten digit dataset.

Preprocess image data for model compatibility and prediction quality.

Evaluate model performance using standard classification metrics.

Serve predictions through a user-friendly Flask web application.

Organize project components using templates, backend logic, and notebook experimentation.

Present a complete mini-ML product rather than only a notebook experiment.
